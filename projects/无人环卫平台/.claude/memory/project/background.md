## 项目定位

**无人环卫平台** — 无人环卫车辆的控制与管理平台。当前已支持**道路清扫车**的监控管理，正在扩展第二类机器人**无人收运车**，覆盖"收运点→中转站"的前端垃圾收运场景。

## 核心差异化

竞品分析表明，行业高度聚焦**清扫环节**，收运环节的无人化控制平台几乎空白。这是差异化切入点：
- 清扫：持续覆盖式（边走边扫），固定片区循环
- 收运：点位收集式（停靠→装载→驶离），固定站点串联（类似公交线路）
- 收运的特殊需求：站点管理、满载逻辑、动态排班、中转站协同

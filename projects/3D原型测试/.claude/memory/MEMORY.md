# 3D原型测试 — 项目记忆索引

## 项目定位

探索在 HTML 原型中集成交互式 3D 模型的可行性，使用 Three.js (r128) 作为渲染引擎。当前产物是一个可交互的 3D 地球查看器，包含自定义 ShaderMaterial、金色飞线、流动光点等高级特效。

## 当前进度

- **阶段**: 技术验证 — 地球原型迭代中
- **最新**: TubeGeometry 管状金带 + 脉冲光环原点 + 萤火虫流动粒子
- **文件**: `prototypes/proto-3d-viewer-v1.html`

## 原型当前状态

- 自定义 ShaderMaterial 地球（日/夜纹理混合 + 海洋 GGX 镜面高光 + 大气 Fresnel 辉光 + 微脉冲）
- 40 条金色飞线从福建龙岩辐射至全球城市（中国 30 条 + 境外 10 条）
- 飞线采用 TubeGeometry 双层结构（additive 辉光管 + 实心核心管）
- 1 个/线萤火虫粒子沿贝塞尔曲线循环运动
- 龙岩原点：白核 + 金色光晕 + 脉冲扩展光环
- 星空背景 + 大气散射壳
- 加载进度条（LoadingManager + head preload）

## 项目定位

**智慧环卫项目** — 环卫业务运营管理平台。覆盖**传统环卫车辆**和**无人环卫车辆**的综合调度、运营管理与数据分析。车辆基础能力（远程驾舱、视频、对讲、路线）由 [[无人环卫平台]] API 提供。

## 当前阶段

### 已上线
- 传统环卫功能：车辆调度、运营管理、数据分析

### 进行中
- 无人环卫车接入功能需求分析（未排期开发）
- 测算工具开发与推广（目标 2026-07）

### 待开始
- 无人清运功能开发（2026-08）
- 无人收运功能接入（2026-10）

## 上下游关系

```
无人环卫平台（厂家端 / 技术底座）
  └── API：远程驾舱、视频、对讲、路线、车辆数据
        └── 智慧环卫项目（运营端 / 业务平台）
              ├── 传统环卫车辆管理
              └── 无人环卫车辆运营
```
